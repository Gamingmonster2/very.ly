# very.ly 📸✨

تطبيق ويب احترافي من صفحة واحدة (SPA) لإزالة خلفيات الصور فوراً وبدقة عالية مباشرة داخل المتصفح، وبخصوصية مطلقة دون الحاجة لأي خادم خلفي (Serverless).

المشروع مبني بالكامل ليعمل كملف مدمج واحد ومُهيّأ ليرفع مباشرة على منصة **Cloudflare Pages**.

---

## 🌐 مميزات المشروع

*   **ذكاء اصطناعي محلي:** معالجة الصور وفصل الخلفية تتم بالكامل داخل متصفح المستخدم باستخدام محرك بايثون (`rembg`).
*   **خصوصية مطلقة بنسبة 100%:** لا يتم رفع صور المستخدمين إلى أي خوادم خارجية؛ مما يضمن الأمان التام للبيانات.
*   **تصميم بسيط وعصري (Minimalist):** واجهة مستخدم متميزة تحمل العلامة التجارية **very.ly** تدعم اللغة العربية (RTL).
*   **بنية تخدم السحاب:** ملف مدمج واحد مجهز للعمل فوراً عند استضافته على **Cloudflare**.
